# CMS Editor Guide

Log in to the Umbraco backoffice and edit the `portfolioHome` page.

## Languages

Edit Swedish content in `sv-SE` and English content in `en-US`. Publish both
cultures before expecting the React site to show the changes.

## Safe Editing

You can edit portfolio text, SEO fields, navigation labels, experience cards,
project/lab copy, contact copy, and Say Hi presentation text.

Do not add Home Assistant details, private URLs, webhook IDs, tokens, or rate
limits to CMS content.

## Three.js Labels

Experience items may provide labels and summaries for known scene keys. Use only
documented scene keys so the career world can map content safely.

## Say Hi

CMS controls copy only. The actual signal to the smart home is controlled by the
secure backend and environment secrets.
