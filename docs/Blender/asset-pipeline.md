# Blender Career World Asset Pipeline

Blender is used for authored miniature geometry: buildings, terrain, forestry props, bevels, normals, materials and optional animation clips. Plain Three.js remains the runtime because the portfolio owns custom controls, raycasting, HTML overlays, aurora, data pulses, fallback behavior and CMS-driven localized content.

## Ownership

- Blender owns source geometry and PBR material values.
- `Assets/Blender/Sources` stores generated `.blend` source files.
- `Assets/Blender/Exports` stores generated local GLB exports and is ignored by Git.
- `Assets/Blender/Previews` and `Assets/Blender/Reports` store local review artifacts and are ignored by Git.
- `HSClient/public/models/career-world` owns runtime GLB delivery.
- Three.js owns renderer setup, camera, controls, lights, interaction, labels, cards, atmosphere, animation lifecycle and fallback.

## Commands

```bash
npm run models:export
npm run models:preview
npm run models:validate
```

Set `BLENDER_EXECUTABLE` only when Blender is not on PATH or in a standard install location.

## Review Workflow

1. Optionally copy current previews into `Assets/Blender/Previews/Before` for local comparison.
2. Rebuild the relevant Python builder under `Assets/Blender/Scripts`.
3. Run `npm run models:export` and `npm run models:preview`.
4. Open the generated PNG previews before accepting the change. These previews are local-only artifacts.
5. Run `npm run models:validate` before starting the client.

## Fallback

The procedural world remains a functional fallback. The GLB environment hides the procedural base only after it validates. Each landmark hides its procedural counterpart only after its own GLB validates. This keeps the hero interactive during local asset work and partial deployment failures.

## Current Limitations

The models are stylized low-poly miniatures generated by Blender Python and reviewed through rendered previews. They are intended for the hero camera, not close architectural inspection. Future manual Blender polish should focus on facade detail, signage typography, better terrain sculpting and custom UV-free material variation.
