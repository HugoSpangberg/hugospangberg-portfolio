# Career World Reference Pack

Copy this folder into the portfolio repository, preferably:

`docs/references/career-world/`

The images are visual references only. They should guide silhouette, facade rhythm,
color, and signature details. The Three.js result should remain a stylized,
procedural miniature matching the existing Nordic Forest Tech career world.

Recommended Codex workflow:

1. Inspect the current working tree.
2. Read `career-world-scene-spec.json`.
3. Inspect all reference images.
4. Preserve hotspot IDs and positions.
5. Refactor landmark builders into focused modules if useful.
6. Implement one landmark at a time.
7. Run build and screenshot checks after each landmark.
8. Tune camera/readability on desktop and mobile.
9. Do not use photo textures or heavy external 3D assets.
