# Södra Codex Pack

Copy the contents of this folder into the root of the portfolio repository.

Expected paths after copying:

- `Assets/Blender/Specifications/sodra-headquarters.json`
- `Assets/CareerWorldReferences/Private/sodra/*.png`
- `CODEX_PROMPT.md`

Run Codex locally from the repository root so it can access Blender installed on the Mac.

The reference photographs are intended as local modelling references. If their reuse rights are unclear, keep the image directory out of Git and add a narrow `.gitignore` rule:

`Assets/CareerWorldReferences/Private/`

The JSON specification and Blender scripts can still be committed.
